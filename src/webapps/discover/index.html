Graph Search Algorithms

Finding the best path in a maze of possibilities

Enroll

Overview

Description

Learn the basics of graph search algorithms, a useful set of algorithms that cover a wide range of applications, from robot pathfinding systems, to route planning and games AI. You will experience the most common implementations in our interactive playground and discuss the benefits and consequences of their use.

Estimated Duration

1 Week

Requisites

No previous skills required

Syllabus

1. Introduction

Discover the many applications of graph search algorithms, find out what graphs are and how can be represented.

2. Graph traversal algorithms

Learn the simplests graph search algorithms: Breadth First Search and Depth First search. Find out how they work and discuss their advantages and disadvantages with yout classmates.

3. Heuristic algorithms

Discover a different kind of search algorithm, the so called: Greedy Best First algorithm. Also learn about heuristics and their impact on algorithm's performance.
